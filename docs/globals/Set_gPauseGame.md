# Set_gPauseGame
```c
// 0x00436730
void Set_gPauseGame(bool toggle, /* optional */ string reason)
```
## Description
```
Minimum Arguments: 1
Maximum Arguments: 2

If you don't sepcify a reason it won't do anything.
And if you specify a reason it does not really matter.
It does not display anything it just pauses / unpauses the game.
```
